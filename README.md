# UPI Payment QR Generator

A simple and modern UPI Payment QR Generator made using **HTML, CSS, and JavaScript**.

This tool allows users to:
- Generate UPI Payment QR Code
- Add UPI ID, Name, and Amount
- Share QR directly using Web Share API
- Beautiful 3D Glassmorphism UI
